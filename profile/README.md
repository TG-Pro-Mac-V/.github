## Overview

TG Pro Advanced is a professional temperature monitoring and fan control application that displays real-time temperature readings from every thermal sensor in your system — CPU, GPU, SSD, memory controller, ambient, and others — alongside fan speed monitoring and manual fan speed control that overrides the automatic cooling system when needed. It provides the thermal visibility and control that power users need to manage heat under heavy workloads, diagnose thermal throttling, and verify that cooling systems are functioning correctly.

Temperature monitoring reads every thermal sensor the hardware provides, displaying current, minimum, and maximum temperatures with historical graphs that show thermal behavior over time. The sensor coverage is comprehensive — not just CPU and GPU, but every sensor the system reports, including some that Apple's own tools do not surface. Real-time display with configurable update intervals shows thermal changes as they happen.

Fan control provides manual speed adjustment that overrides the automatic fan management system, enabling users to increase cooling during sustained heavy workloads or reduce fan noise during light tasks where the automatic system runs fans higher than necessary. Custom fan rules set temperature-based speed curves that provide quieter operation at low temperatures while ensuring adequate cooling under load. Fan control presets save configurations for different usage scenarios.

Diagnostics verify that all fans are functioning correctly and that temperature sensors are reporting plausible values, identifying hardware problems that might cause overheating before they lead to thermal throttling or system shutdown. The diagnostic tests spin fans to verify operation and compare sensor readings against expected ranges.

For users who push their hardware with demanding workloads and want visibility into thermal behavior with the ability to optimize cooling, TG Pro Advanced is the thermal management tool that provides the temperature data and fan control that informed hardware management requires.
